🏦 Java Banking System (Learning Project)
📌 About

This is a simple banking system I built while learning Java step by step.
I didn’t build everything at once — I improved it version by version to understand concepts properly.

🧠 What I Learned
Basic Java programming
How to use classes and objects (OOP)
Using loops and switch for menu systems
Handling errors using try-catch
Saving data using files
Moving from file storage to database (SQLite + JDBC)
🔄 My Progress
V1 → Basic deposit and withdraw
V2 → Added menu
V3 → Created BankAccount class
V4 → Added exception handling
V5 → Saved data using file
V6 → Switched to SQLite database
▶️ How to Run

Compile:

javac -cp ".;sqlite-jdbc-3.x.x.jar" *.java

Run:

java -cp ".;sqlite-jdbc-3.x.x.jar" Main
